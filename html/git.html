Git

  • git init - инициализирует новый репозиторий GIT и начинает отслеживание существующего каталога.
  • dir/ ah - проверка наличия скрытых файлов в репозитории.
  • git status - проверка индексации изменений и возможность увидеть файлы, которые не отслеживаются Git.
  • git commit - делает для проекта снимок текущего состояния изменений, добавленных в раздел проиндексированных файлов.
  • git push - позволяет передать изменения из локального репозитория в удаленный.
